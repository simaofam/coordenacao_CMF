⚙️ Sincronização entre Dispositivos

Estado actual: A carregar...
Como ligar a sincronização (configuração única de 5 minutos):
  1. Abre https://console.firebase.google.com (com a tua conta Google)
  2. Clica "Add project" → nome qualquer → Continue → desativa Analytics → Criar
  3. No menu lateral esquerdo: Build → Realtime Database → Create Database
  4. Escolhe localização europe-west1
  5. Seleciona "Start in test mode" → Enable
  6. Copia o URL no topo da página (algo como https://meu-projeto-default-rtdb.europe-west1.firebasedatabase.app)
  7. Cola aqui em baixo e clica Ligar
  8. Repete o passo 7 no telemóvel e no PC (mesmo URL para todos)
⚠ Em "test mode" o acesso é livre por 30 dias. Para uso prolongado, vai a Realtime Database → Rules e altera as regras para true permanente, ou implementa autenticação. Para uma escola privada com poucos utilizadores isto é suficiente.
